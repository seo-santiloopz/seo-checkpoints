# Twitter


### Twitter specific `meta` tags
```html
<head>
	<meta name="twitter:title" content="title of the card">
	<meta name="twitter:site" content="twitter handle">
	<meta name="twitter:card" content="type of twitter card that we want to generate (i.e. summary)">
	<meta name="twitter:image" content="imgUrl">
	<meta name="twitter:text:description" content="desc">
	<meta name="twitter:description" content="long desc">
</head>
```
